## Claude Pokemon Watch

This is an AI bot that captures screenshots of Claude AI playing pokemon and write a short blurb about what it's seeing

### Setup

1. Add these enviroment variables to a .env file
	```
	TWITCH_CHANNEL=
	INTERVAL_MINUTES=
	OPEN_ROUTER_API_KEY=
	OPENROUTER_MODEL=
	X_API_KEY=
	X_API_SECRET=
	X_ACCESS_TOKEN=
	X_ENABLED=true
	```
